# Contributing to sndwrks documentation

Thanks for helping improve our docs! This repo is public so you can read the
documentation and propose changes. Contributions of all sizes - fixing a typo,
clarifying a step, or adding a missing detail - are welcome.

## How to contribute

1. **Open an issue** if you've found a problem but aren't sure how to fix it,
   or want to discuss a larger change first.
2. **Open a pull request** for direct edits:
   - Fork the repo and create a branch off `main` (e.g. `main.fix-typo`).
   - Make your change. For content edits, see the
     [documentation style guide](https://docs.sndwrks.com/reference/doc-style).
   - Run `npm run dev` to preview locally, or `npm run build` to confirm it builds.
   - Open a PR with a short description of what changed and why.
